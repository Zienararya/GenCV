# GENCV
projek ini bekerja dengan cara mengenerate CV yang dimasukkan diubah ke dalam bentuk pdf sebagai output File
## SYSTEM REQUIREMENTS:
- node v20
- php 8.2
- composer
## TESTING MODE:
- git clone https://github.com/Zienararya/GenCV.git
- php artisan install
- npm install
- php artisan seed migrate --seed
- php artisan serve

## STACK
- Laravel 11
- tailwind css
